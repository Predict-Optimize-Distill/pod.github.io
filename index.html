TL;DR: POD uses a self-improving cycle for 4D object understanding from an object scan and single monocular video.

For novel objects, humans can resort to long-form inspection to build intuition of how object moves in 3D space. The more we observe the object, the better we get at predicting its 3D state immediately. Similarly, POD achieves better 4D object understanding with increasing observation time.

4D Object Part Understanding

POD takes in 1) a multi-view object scan and 2) a monocular demonstration video. By creating part-aware 3D representations using GARField (parts, toggle for clusters) and DINOv2 (tracking SE3 pose), POD achieves 4D object part tracking.
